🧠 Portfolio Website (Initial Commit)
🚀 Overview

This repository contains the initial version of my personal portfolio website — currently featuring the homepage section only.
The full site is under active development and will include additional sections such as Projects, About Me, and Contact once completed.

This commit represents my first deployed milestone, showing the base structure, theme system, and initial design approach.

⚙️ Tech Stack

Frontend Framework: React (Vite)

Styling: Tailwind CSS

Package Manager: npm

Version Control: Git & GitHub

Development Tools: VS Code

Other Tools I Use: Proxmox, VMware, UTM, Tailscale (WireGuard-based VPN), Adobe Photoshop

💡 Current Features

Responsive homepage layout

Hero section with animation (GIF integrated)

Dark/Light mode toggle

Minimalist React component structure

Tailwind-based responsive styling

🧩 Next Steps / Upcoming Features

Projects showcase page

About Me & Skills section

Contact form integration

Deployment on Vercel or GitHub Pages

Blog/learning journal section

🧰 Run Locally
git clone https://github.com/Olci2/Portfolio.git
cd Portfolio
npm install
npm run dev


Then open your browser and go to:

http://localhost:5173
